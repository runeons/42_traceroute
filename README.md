# 42_ft_traceroute
    
#### Overview
Following ft_ping, this is a 42 project about recoding the traceroute command.

#### Options
-    -h, --help
-    -q, --nqueries
-    -m, --max-hops
-    -w, --wait
-    -z, --sendwait
-    -p, --port
-    -f, --first-ttl
-    -n, --no-dns
-    -v, --verbose


#### Information
- requires superuser privileges
- functional on Debian GNU/Linux 12

#### Usage
    make && ./ft_traceroute -h
